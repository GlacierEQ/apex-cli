# APEX CLI — Autonomous Perplexity EXecution Engine

> Terminal-native AI with APEX legal intelligence, RICO/§1983 context, and full GlacierEQ stack awareness.

## Install

```bash
npm install
chmod +x apex.js
npm link     # makes 'apex' and 'perplexity' available globally
```

## Quick Start

```bash
export ANTHROPIC_API_KEY=sk-ant-...
apex
```

## Commands

| Command | Description |
|---------|-------------|
| `/legal [topic]` | Deep legal analysis — Hawaii statutes + federal law |
| `/case` | Full Case 1FDV-23-0001009 tactical brief |
| `/code [task]` | Production code generation |
| `/apex [task]` | APEX orchestration operations |
| `/search [q]` | Web intelligence brief |
| `/context [mode]` | Switch: `legal` / `code` / `apex` / `general` |
| `/model [name]` | Switch AI model |
| `/key [sk-ant-...]` | Set API key |
| `/save [file]` | Save session |
| `/load [file]` | Load session |
| `/history` | Recent query log |
| `/clear` | New session |
| `/help` | Command reference |
| `/exit` | Exit (auto-saves) |
